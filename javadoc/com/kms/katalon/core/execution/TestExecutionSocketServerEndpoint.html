Package: com.kms.katalon.core.execution

[Java] Class TestExecutionSocketServerEndpoint

      • com.kms.katalon.core.execution.TestExecutionSocketServerEndpoint 

      • public class TestExecutionSocketServerEndpoint
extends Endpoint

        This is the end point that receives the captured MHTML from browser extension's socket client. The important method is TestExecutionSocketServerEndpoint.handleMhtml which reads the byte array returned by the socket client and separates it into the two parts (hashed test case ID + MHTML file)

          • public void waitForMhtmlHandlingThread()

            This method waits within 5 seconds for a thread that handles MHTML to start. This logic is needed because MHTML should be sent from browser's extension after Katalon sends EXECUTION_END command, but Katalon's thread will continue after sending the command which may terminate before receiving the MHTML

            After 5 seconds, if a thread does start, it waits until the thread is done.
